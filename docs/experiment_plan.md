# 当前论文实验计划

## 核心主张

在车身真实颜色清晰可见的条件下，多模态模型总体表现出明显的视觉优先性。错误文本提示通常不足以覆盖视觉证据，只有在特定强误导性提示下，部分模型才会出现有限的语言主导偏差。

## 研究问题

- RQ1：在图文冲突条件下，模型输出主要由视觉证据还是语言提示驱动？
- RQ2：弱提示、错误断言、预设前提等不同提示强度和设计，对模型偏离视觉真相的影响有何差异？
- RQ3：`LLaVA-1.5-7B`、`Qwen2-VL-7B-Instruct`、`InternVL2-8B` 在该任务上的鲁棒性有何不同？

## 正式实验对象

- 正文正式实验对象是 `data/balanced_eval_set/final_manifest.csv`
- 该评测集为 6 色平衡集，每色 50 张，总计 300 张
- Stanford Cars 严格清洗子集仅作为数据来源和构建动机
- Stanford-only 流程不再作为正文主实验主线，只保留为 appendix sanity check

## P0 必做

- `C0` 基线：验证三模型在无误导提示时的基本识别能力
- `C0-C4` 主实验：输出冲突一致率、忠实率、拒绝率、95% CI
- 三模型统一比较：`LLaVA-1.5-7B`、`Qwen2-VL-7B-Instruct`、`InternVL2-8B`

## P1 建议补充

- `A1/A2` 辅助实验：测试受限回答条件下的提示服从性
- 代表性案例与错误案例整理：用于 Discussion 或附录

## P2 可选

- Stanford-only 轻量 sanity check
- 提示措辞小样本变体
- 扩大样本量、增加模型或任务

## 明确不再投入

- 决策边界、阈值、LDR、语言主导率公式等理论包装
- 多任务属性冲突泛化
- 与当前主结论无关的探索脚本、debug 流程和历史中间产物
